Expandable Card

### 🚀 Características:
* **Diseño Responsivo:** Centrado absoluto con Flexbox.
* **Animaciones CSS:** Uso de `max-height` y `opacity` para un despliegue suave del contenido.
* **Estética Gamer:** Inspirado en la interfaz de usuario de Warframe, utilizando una paleta de colores "Oro Prime" y fondos oscuros.

### 🛠️ Tecnologías:
* HTML5
* CSS3 (Transitions, Hover states, Flexbox)
* JavaScript (Event Listeners básicos)
